# my-blog
动手写的一个简易博客（菜鸟）

## 运用到的技术
后端：node、express、mongoose
<br/>
前端：vue-cli3、elementui、axios、vue-router

## 运行
1、cd my-blog<br/>
2、npm install<br/>
3、cd client<br/>
4、npm install<br/>
5、创建mongoose数据库<br/>
6、npm run dev
